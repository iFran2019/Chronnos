# Privacy Policy

**Last Updated:** July 14, 2026

This Privacy Policy explains how CHRONNOS ("the Bot") processes information while providing its features.

## Information We Process

To provide its functionality, the Bot may process the following information:

- Discord User IDs
- Discord Server IDs
- Discord Channel IDs
- Discord Role IDs
- Discord Message Content (only when required for specific features such as the AFK system)
- Voice channel information required for music playback

## How We Use Information

The information processed by the Bot is used solely to provide its features, including but not limited to:

- Music playback and queue management
- AFK status management
- Command execution
- Server-specific configuration
- User information commands

## Message Content

When Message Content Intent is enabled, message content is processed only to provide features that depend on regular messages, such as:

- Automatically removing a user's AFK status when they send a message
- Detecting mentions of AFK users to notify other members

Message content is processed only for these purposes and is not permanently stored.

## Data Storage

CHRONNOS does not permanently store:

- Message content
- User conversations
- Voice data
- Presence data

Any data processed is used only for the duration necessary to provide the requested functionality.

## Data Sharing

CHRONNOS does not sell, rent, or share user data with third parties.

## AI and Machine Learning

CHRONNOS does not use any user data or message content to train artificial intelligence or machine learning models.

## Security

Reasonable measures are taken to protect any information processed by the Bot during operation.

## Your Rights

If you have questions regarding this Privacy Policy or believe the Bot is processing data incorrectly, you may contact the developer.

## Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of CHRONNOS after changes indicates acceptance of the updated policy.
