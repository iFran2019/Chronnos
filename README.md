# Privacy Policy

**Last Updated:** July 14, 2026

This Privacy Policy explains how CHRONNOS ("the Bot") processes and uses information while providing its features.

## Information We Process

To provide its functionality, CHRONNOS may process:

- Discord User IDs
- Discord Server IDs
- Discord Channel IDs
- Discord Role IDs
- Discord Message Content (only when required for specific features, such as the AFK system)
- Voice channel information required for music playback

## How We Use Information

The information processed by CHRONNOS is used exclusively to provide its features, including but not limited to:

- Music playback and queue management
- AFK status management
- Command execution
- User information commands
- Server-specific functionality

## Message Content

When the Message Content Intent is enabled, message content is processed only when necessary to provide features that depend on normal user messages.

For example, CHRONNOS may:

- Automatically remove a user's AFK status when they send a message.
- Notify users when they mention someone who is currently marked as AFK.

Message content is processed only for these features and is **not permanently stored**.

## Data Storage

At this time, CHRONNOS does **not** permanently store:

- Message content
- User conversations
- Voice data
- Presence data
- Personal user information

In the future, CHRONNOS may store limited information that is explicitly created or configured by users or server administrators, such as:

- Saved playlists
- Server configuration
- User preferences
- Feature settings

Any stored information will be used solely to provide the requested functionality.

## Data Sharing

CHRONNOS does not sell, rent, or share user data with third parties.

## Artificial Intelligence

CHRONNOS does **not** use Artificial Intelligence (AI) or Machine Learning (ML) to process, analyze, or generate responses based on user data.

User data, including message content, is **never** used to train AI or Machine Learning models.

## Security

Reasonable measures are taken to protect any information processed by CHRONNOS during operation.

## Your Rights

If you have any questions regarding this Privacy Policy or believe CHRONNOS is processing data incorrectly, you may contact the developer.

## Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected by updating the "Last Updated" date at the top of this document.

Continued use of CHRONNOS after any changes constitutes acceptance of the updated Privacy Policy.

## Open Source Framework

CHRONNOS is built upon the open-source framework **BotMaster**, available at https://github.com/iFran2019/BotMaster.

The framework provides the core architecture of the bot. CHRONNOS includes its own features, configuration, and functionality built on top of this foundation.
